# Neural Gu-Zhen

**神经蛊阵** — 基于多链 DNA 磁吸匹配的 AI Agent 记忆生态系统。

### 核心创新
- DNA 即坐标：节点自动获取结构空间坐标，无需标签
- 5 链磁吸匹配：domain/intent/entity/pattern/context 组合投票
- 双向 DNA 规制：执行意图与用户原则自动碰撞
- 生态三算子：能量枯竭、同类吞噬、异种杂交

### Benchmark
LongMemEval 零指令零 token 零检索条件下 81.5% 命中率。
